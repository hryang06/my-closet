웹 페이지 URL을 통해 의류 제품 정보를 확인하고 신체 사이즈를 기반으로 사이즈를 추천하는 서비스.

## 사용자 흐름

1. **입력** — URL 붙여넣기, 지우기 버튼으로 리셋
2. **변환** — 본문에서 의류 제품 정보를 추출
   - 정보: 제품 이름, 브랜드, 사이즈
3. **결과 확인** — 제품 제목 / 브랜드 헤더 / 사이즈 테이블로 표시
4. **사이즈 추천** - 사이즈 정보 아래 사이즈 추천 버튼
   - 버튼 동작
        - 신체 사이즈 정보가 없는 경우, 신체 사이즈 정보 입력 창으로 입력 유도
        - 신체 사이즈 정보가 있는 경우, 사이즈 추천

## 프롬프트
- 프리셋: 요약해줘 / 한국어로 번역해줘 / 쉽게 설명해줘
- 직접 입력: 프리셋 옆 "직접 입력" 선택 시 텍스트 인풋 노출, 1회용(저장하지 않음)
